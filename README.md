
# To Do List

Un Esercizio fatto con start2Impact per imparare Angular

## Installation

Clone repository

<h3>Install dependencies</h3>

```bash
  npm install 
```

<h3>Run Angular server (http://localhost:4200)</h3>

```bash
  ng serve
```
<h3>Run the JSON server (http://localhost:4000)</h3>

```bash
  npm run server 
```
